# Iron Team — V1 Cloud (Supabase)

## Lancer en local
1) Installer Node.js (LTS)
2) Dans le dossier :
   npm install
   npm run dev

## Important (lien magique)
Supabase → Authentication → URL Configuration
- Site URL : http://localhost:5173
- Redirect URLs : ajouter http://localhost:5173

Quand tu déploies, ajoute aussi l’URL du site (Netlify/Vercel) dans Redirect URLs.
